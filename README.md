# forCup
Esta es una herramienta para elegir el formato del videojuego "Cuphead" por si quieres jugar en una CRT o en su formato original.

Proximamente se lanzara una version para Windows por ahora solo esta disponible en Linux


## INSTALACION

Instalas GIT

    # sudo apt install git

Instalas el repositorio

    # git clone https://github.com/bramicu13/forCup

Cambias el archivo "RUTA"

    porfavor-incluya-la-ruta-para-la-carpeta-de-cuphead --> ejemplo: /home/usuario/.steam/Steam/steamapps/compatdata/268910/pfx/drive_c/users/steamuser/AppData/Roaming/Cuphead/Cuphead_Data/Managed/

Y LISTO!

Igual para evitar problemas busque por si mismo la carpeta ¡IMPORTANTE QUE LA RUTA TERMINE EN LA CARPETA "MANAGED" SIEMPRE!
